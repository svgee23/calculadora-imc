#🧍Calculadora de IMC en Python

Este programa solicita datos del usuario como nombre, estatura y peso corporal, ademas que calcula el Indice de Masa Corporal (IMC).
muestra la clasificacion del IMC segun rangos de salud definidos.

#🛠️ Funcionalidades
- Solicita nombre, estatura y peso del usuario
- Calcula el IMC
- Muestra la clasificacion del IMC (ej: delgadez severa, normal, sobrepeso, etc.)
- Clasificacion adicional segun la estatura del usuario

#🔠 Lenguaje utilizado
- Python

#▶️ Como puedes ejecutar este programa

1. Asegurate de tener Python instalado
2. Descarga el archivo o clona el repositorio
3. Ejecuta el programa:

python calculadora_IMC.py
